Help & Getting Started
📦 Repository

Full documentation, setup instructions and community results are available on GitHub:

github.com/Rogala/SageAttention-Benchmark-Viewer
🐍 Benchmark Script

To generate your own results, download the script and run it locally:

⬇ sage_bench_models.py

Run command:

python sage_bench_models.py --warmup 20 --iters 100

The script automatically saves the result as a JSON file named after your GPU (e.g. RTX-5060_Ti-16GB.json) in the current directory.
